## Description: <br>
Learns your tool preferences while staying capable of using anything. Adapts to your stack. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and agent users use this skill to record and apply tool preferences, defaulting to familiar tools while still allowing better alternatives when they materially improve the task. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Stored preference sections could accidentally contain sensitive project details or secrets. <br>
Mitigation: Keep Stack, Preferences, Open To, and Avoid entries limited to non-sensitive tool preferences; do not store tokens, passwords, private customer data, or sensitive project details. <br>
Risk: Suggestions involving unfamiliar tools could lead to paid services, software installation, public posting, or important data changes. <br>
Mitigation: Ask for confirmation before using unfamiliar tools for paid services, software installation, public posting, or changes to important data. <br>
