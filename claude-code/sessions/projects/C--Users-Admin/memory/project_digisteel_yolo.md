DigiSteel-YOLO is Hazem's graduation project for the Digilians (MCIT) Specialized Diploma in Applied AI & Data Analytics. Team of 5 students led by Hazem Elerefy, supervised by Dr. Tarek Ghoneimy. ~12 weeks remaining as of late May 2026.

**Why:** Build a competitive steel defect detector based on YOLOv11n and publish a systematic robustness evaluation framework — filling a gap where no prior work tests YOLO detectors under real-world image degradations.

**How to apply:** Hazem wants to beat 11 reference papers (top is P10 KDM-YOLO at 95.4% mAP@0.5). Current baseline is 77.9%. The approach (GhostConv + Inner-WIoU) is insufficient to close the 17.5% gap. Major architectural additions needed. Related: [[user-hazem-profile]]
